# Aanvikshiki — Full Redesign Spec

> IBM-inspired enterprise B2B redesign with full-width image blocks,
> bold typographic hierarchy, and unified component system.

---

## Design Direction: IBM Carbon — Adapted

Reference: ibm.com (Carbon Design System aesthetic)
- Bold, square geometry — no decorative rounding
- Typography-first with IBM Plex Sans
- Full-width alternating dark/light sections
- Large background photography throughout
- Structured grid, no visual clutter
- Authority through restraint, not decoration

---


## 4. Component System Overhaul

### Remove Entirely
- `antd` package (Tabs, Progress, Statistic, Drawer, Menu, FloatButton, Button)
- Estimated bundle savings: ~2MB

### Replace With
| Ant Design Component | Replacement |
|---------------------|-------------|
| `Tabs` | Custom `<TabGroup>` using Radix or plain state |
| `Progress` | Custom `<MetricBar>` (Tailwind-native) |
| `Statistic` | Plain div with IBM Plex Mono number |
| `Drawer` | Radix `Dialog` configured as left-slide Sheet |
| `Menu` | Radix `NavigationMenu` (already installed) |
| `FloatButton.BackTop` | Custom back-to-top `<button>` |
| `AntButton` (menu trigger) | shadcn `Button` variant="ghost" |

### Design Token Updates
- Remove border-radius from all cards: `rounded-lg` → remove or `rounded-none`
- Section padding: increase to `py-28 md:py-36` for IBM-style breathing room
- Add `max-w-screen-xl` option for wider sections
